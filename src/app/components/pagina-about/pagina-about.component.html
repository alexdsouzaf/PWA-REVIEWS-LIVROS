Do que se trata essa página?

Feito para estudo, é um WebApp para anotações de reviews/resenhas de livros para uso pessoal do usuário que deseja registrar pensamentos após finalizar suas leituras.

O código fonte do projeto pode ser encontrado nesse repositório do github, o mesmo responsável pela página estar no ar.

Essa versão disponível armazena todos os dados no Local Storage do próprio navegador e conta com suporte para PWA, permitindo ser instalado como um app em celulares e computadores.
